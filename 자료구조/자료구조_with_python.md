data structure with python
=============================

- array(배열)
  - 정의 : 배열에는 같은 종류의 데이터들이 순차적으로 저장되어, 값의 번호가 곧 배열의 시작점으로부터 값이 저장되어 있는 상대적인 위치가 된다.
  - 선언 : ```arr = [1,2,3]```
  - 장점:
    - 빠른 접근 가능
      - 첫 데이터의 위치에서 상대적인 위치로 데이터 접근(인덱스 번호로 접근)
  - 단점:
    - 데이터 추가/삭제의 어려움
      - 미리 최대 길이를 지정해야 함
